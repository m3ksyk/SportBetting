Here you can learn about using data feed we provide for our users. To use this feed you need to use your unique api key You can get you apikey HERE Methods: all method urls work by this scheme: URL : api/{method name}?apikey={your_apikey} List of methods: get leagues: GET /api/get-leagues returns supported football leagues get sports: GET /api/get-sports returns supported sports get teams: GET /api/get-teams returns supported teams with their results and current standing get live games: GET /api/get-live-games returns live games get scheduled games: GET /api/get-scheduled-games returns scheduled games get game results: GET /api/get-games-results returns the games results;
