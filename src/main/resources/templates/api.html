Here you can learn about using data feed we provide for our users. To use this feed you need to use your unique api key

get your api key

Methods: all method urls work by this scheme: URL : api/{method name}?apikey={your_apikey}

List of methods:

  1. get leagues:

    GET /api/get-leagues

    returns supported football leagues

  2. get sports:

    GET /api/get-sports

    returns supported sports

  3. get teams:

    GET /api/get-teams

    returns supported teams with their results and current standing

  4. get live games:

    GET /api/get-live-games

    returns live games

  5. get scheduled games:

    GET /api/get-scheduled-games

    returns scheduled games

  6. get game results:

    GET /api/get-games-results

    returns the games results;
